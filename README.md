# Teleprompter Web App

一個基於 Web 的簡單提詞器應用程式，專為內容創作者、演講者和專業錄製者設計。無需下載安裝，打開瀏覽器即可使用。

## 線上演示 (Demo)

[https://teleprompter.zeabur.app/](https://teleprompter.zeabur.app/)

## 功能特色

- **純前端運作**：基於 HTML5, CSS3, Vanilla JS，無需後端伺服器，安全且快速。
- **直覺編輯**：
    - 直接在網頁上貼上或編輯講稿。
    - **快速操作**：提供「貼上」與「清除」按鈕，快速填入或清空內容 (含防呆確認)。
- **靈活控制**：
    - **字體大小**：可隨意調整文字大小以適應不同閱讀距離。
    - **滾動速度**：可調整文字向上滾動的速度。
    - **手勢微調**：播放時可直接在畫面上滑動手指或滾輪，手動微調文字位置。
    - **視線引導**：沉浸模式下提供水平視線引導線。
    - **邊距調整**：可自訂左右邊距。
- **專業錄製支援**：
    - **畫面反轉 (Flip)**：支援「上下反轉」功能，適用於搭配分光鏡的專業提詞機硬體。
    - **倒數計時**：播放前提供 3 秒倒數緩衝，讓您有時間準備 (預設開啟)。
    - **全螢幕模式**：一鍵進入沉浸式全螢幕播放，設定面板自動隱藏，確保無干擾。

## 如何使用

1. **開啟應用**：直接用瀏覽器打開 `index.html` 檔案。
2. **輸入腳本**：在文字區域輸入或是貼上您的演講稿。
3. **調整設定**：
    - 使用滑桿調整字體大小和滾動速度。
    - 若使用提詞機硬體，點擊「Flip Vertical」按鈕將畫面上下顛倒。
4. **開始播放**：
    - 點擊「Play」按鈕開始滾動。
    - 畫面會進入全螢幕模式。
    - 可以隨時點擊畫面暫停/繼續，或使用 ESC/退出按鈕離開全螢幕。

## 技術架構

- **HTML5**: 語意化標籤結構。
- **CSS3**: 使用 Flexbox 版面配置，CSS Transforms 實現畫面反轉。
- **JavaScript (ES6+)**: 處理滾動邏輯與介面互動，不依賴任何外部框架。
